## 📖 Overview

A fully static portfolio webpage developed with vanilla HTML, CSS, and JavaScript. It features a custom JavaScript typewriter animation on page introduction and CSS keyframe animations applied across the page sections for dynamic visual transitions. The site demonstrates layout, styling, and UI interactions. Actual personal details have yet to be added; dynamic data fetching may be applied in the future.

## 🚀 Features

- **Responsive Navigation Bar** – Allows quick access to each section for smooth browsing.
- **Animated Section Reveals** – Uses CSS keyframes and JavaScript to display content with engaging transitions.
- **Contact Button** – Directs users to an external contact form for easy communication.

## 🛠️ Tech Stack

- **Frontend (UI):** HTML, CSS, JavaScript
